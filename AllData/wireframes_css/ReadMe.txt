This folder contains the structure of the wireframes in the form of file-folder system resembling the same which was made using figma
Each wireframe is represented by a folder which consists of other components again as files or folders(a hierarchical system).
Each component shall be containing its own properties as ".Properties.txt" file and the sub-components as its sub-folders/files.

The link to the figma project is
https://www.figma.com/file/ED8Z53fCtiwwGF2p36KmSF/GSoC_Web?node-id=1%3A2
(can be viewed on web using figma website or the desktop app)
